Qui nous sommes

Nous sommes des développeurs Web qui se spécialisent en accessibilité. Nous évaluons les sites Web et applications d’EDSC pour s’assurer qu’ils respectent les Normes Web du Gouvernement du Canada. Nous sommes également impliqués dans plusieurs projets différents reliés à l’accessibilité. En voici quelques-uns :

  • Réunions communautaires sur l’accessibilité à Ottawa et Montréal
  • Contribution au projet de la Boîte à outils de l’expérience Web
  • Création de modules Firefox pour l’accessibilité
  • Développement et maintenance de la Solution de gabarits à déploiement centralisé (SGDC)
  • Membre actif du Groupe de travail sur l’accessibilité

Notre travail

  • Faire des évaluations de la conformité sur les actifs Web d’EDSC
  • Fournir une expertise et des conseils dans le domaine de l'accessibilité
  • Appuyer les développeurs Web

Normes Web du SCT

  • Accessibilité des sites Web
  • Facilité d’emploi des sites Web
  • Interopérabilité des sites Web
  • Optimisation des sites Web et des applications pour appareils mobiles

Norme sur l’accessibilité des sites Web

  • S’applique sur toutes les pages Web qui s’adressent au public
  • Veiller à ce que chaque page Web satisfasse aux exigences de conformité à WCAG 2.1 niveaux A et AA

Qu’est-ce que WCAG?

  • Ensemble de recommandations pour rendre les contenus Web plus accessibles, principalement pour les personnes handicapées — mais aussi pour tous les agents utilisateurs, y compris les appareils très limités, tels que les téléphones mobiles
  • Norme internationale du W3C

Les principles de WCAG

  • Perceptible – L'information et les composants de l'interface utilisateur doivent être présentés à l'utilisateur de façon à ce qu'il puisse les percevoir
  • Utilisable – Les composants de l'interface utilisateur et de navigation doivent être utilisables
  • Compréhensible – Les informations et l'utilisation de l'interface utilisateur doivent être compréhensibles
  • Robuste – Le contenu doit être suffisamment robuste pour être interprété de manière fiable par une large variété d'agents utilisateurs, y compris les technologies d'assistance

Qui est affecté?

Suivre ces lignes directrices rendra le contenu accessible à un plus vaste éventail de personnes handicapées, y compris les personnes :

  • aveugles et à vision réduite,
  • sourdes et ayant une perte de l'ouïe
  • ayant des troubles d'apprentissage,
  • ayant des limitations cognitives
  • ayant une mobilité réduite,
  • ayant un trouble de la parole
  • ayant des problèmes de photosensibilité ou
  • ou une combnaison de ceux-ci

Suivre ces lignes directrices rendra aussi votre contenu Web plus accessible aux utilisateurs en général.
